## 구조체
### 필요성
모아보기 가능
### 배열과의 차이점
- 배열: 같은 타입의 변수
- 구조체: 서로 다른 타입의 변수
### 구조체 선언
```cs
struct student{
    int number; 
    char name[10]; 
    double grade; 
}; //세미콜론 필요!

struct student s1 = {24, "Kim", 4.3}; //선언방식

s1.grade = 3.8; // 구조체 멤버 접근
```
